## [5.0.0-beta.1] - 2026-02-05

### Added
- 65+ UI components covering forms, buttons, feedback, layout, navigation, and data display
- Smart Tailwind class merging via `UI::classes()` with conflict resolution
- Full Livewire support with automatic `wire:model` binding
- Alpine.js integration for interactive components
- Comprehensive test suite with 1711 tests and 3291 assertions
- Dark mode support for all components
- Tailwind 4 theming with CSS variables
- Multi-language support (EN, NL)

### Components
- **Form**: Input, Textarea, Select, Checkbox, Radio, Switch, Toggle, Slider, File Upload, Tags Input, OTP Input, Datepicker, Timepicker
- **Buttons**: Button, Button Group, Action Bar
- **Feedback**: Alert, Badge, Toast, Progress, Skeleton
- **Layout**: Card, Modal, Dropdown, Tabs, Accordion, Section, Box
- **Navigation**: Nav, Breadcrumbs, Pagination, Steps
- **Data Display**: Table, List, Definition List, Avatar
- **Specialty**: Editor, Code Editor, Pricing, App Switcher, Theme Switcher
