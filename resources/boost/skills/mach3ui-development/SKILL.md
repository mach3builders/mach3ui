# Mach3UI Development

## Overview

This skill covers best practices and guidelines for developing user interfaces using the mach3ui package. It includes conventions for Blade/HTML structure, component usage, translations, and more to ensure consistency and maintainability across the application.

## When to activate
Activate this skill when working on any user interface development tasks that involve the mach3ui package, including creating new views, modifying existing ones, or implementing new UI components.

### Component Usage
basic component usage:
```
<ui:button variant="primary">Create</ui:button>
```

### With properties
```
    <ui:input label="Email" type="email" wire:model="email" :disabled="$isDisabled" />
```
